# AstroSage-Call-Centre-Analysis-

## Problem
AstroSage received a ₹1 crore investment to optimize its call center. As an analyst, I analyzed historical performance data to recommend how to allocate this budget for maximum efficiency, customer satisfaction, and profitability.

## Key Findings
- 73% of users are dissatisfied
- 71% of chats fail | 41% of calls complete
- 129 gurus available, but 21 took zero calls
- Average utilization: only 1.9 calls per guru per day
- Top 10 gurus handle 50% of all calls

## Recommendations
- **70%** on fixing chat and call platform (root cause)
- **15%** on training mid-performing gurus (ratings 2.5–4.0)
- **15%** on replacing bottom 10% performers + retaining top gurus
- **Zero** on new hiring (we have idle capacity)
